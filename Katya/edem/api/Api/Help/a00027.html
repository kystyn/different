Detailed Description

The IFieldManagerApi_1_0 provides basic access to the field manager via a series of query methods.

NAME: Field Manager ID: EApiId::eFieldManager (1) VERSION: 1.0

At this time only read methods are supplied.

pure virtual

Returns a Field API instance for the specified field ID.

The Field Manager is free to return any minor revision equal to or later than the specified value, as long as the major revision matches.

All APIs returned by this method should be released back to the API Manager via its release() method once you have finished with them. Do not delete API instances directly or it may result in system instability and data corruption.

Parameters
