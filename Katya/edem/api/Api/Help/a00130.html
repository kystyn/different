EXPORT_MACRO int GETFACTINTERFACEVERSION ( )

Returns the version of the plugin interface implemented by this plugin file.

TO BE IMPLEMENTED BY END USER

The version number must be packed in to a single 32 bit value. Consider the following example code:

static const int INTERFACE_VERSION_MAJOR = 0x02; static const int INTERFACE_VERSION_MINOR = 0x00; static const int INTERFACE_VERSION_PATCH = 0x00;

return (INTERFACE_VERSION_MAJOR << 16 | INTERFACE_VERSION_MINOR << 8 | INTERFACE_VERSION_PATCH);

Returns
Interface version packed in to a single 32bit value
