Detailed Description

The IFieldApi_1_0 provides basic access to field data via a series of query methods.

NAME: Field ID: EApiId::eField (2) VERSION: 1.0

At this time only read methods are supplied.

Queries a scalar field for the interpolated field value at a given point in EDEM's global co-ordinate system.

EDEM will map from the supplied point to the fields internal co-ordinates using the orientation set in the creator's field manager dialog.

Queries a vector field for the interpolated field value at a given point in EDEM's global co-ordinate system.

EDEM will map from the supplied point to the fields internal co-ordinates using the orientation set in the creator's field manager dialog.

For a 2D vector field, results in one dimension (depending on the plane's orientation) will always be zero

Parameters
pointXX Co-ordinate of the query point
pointYY Co-ordinate of the query point
pointZZ Co-ordinate of the query point
numberOfPointsHow many points to use when calculating the interpolated result
interpolationTypeThe type of interpolation to use.
resultX[RETURN VALUE] X dimension of the vector field strength
resultY[RETURN VALUE] Y dimension of the vector field strength
resultZ[RETURN VALUE] Z dimension of the vector field strength
Returns
true if an interpolated value could be calculated, else false.
